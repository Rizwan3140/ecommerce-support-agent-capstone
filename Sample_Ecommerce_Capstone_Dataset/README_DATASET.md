# E-Commerce Support Capstone Dataset (Sample)

This dataset is intentionally **generic** and **test-only**. It does not represent any real brand.

## Files
- `FAQ.md`: FAQs for customer support (RAG knowledge base)
- `Policies.md`: Policies for shipping, returns/refunds, cancellations, warranty, payments, privacy (RAG knowledge base)
- `Products.json`: Sample product catalog (tool or retrieval source)
- `customer_queries_sample.csv`: Sample queries with intents for testing
- `README_DATASET.md`: This guide
